# Kalour Beauty Empire: UI/UX Professional Upgrade Plan

## Overview

The current website design is too basic and uses a dark theme with less professional typography. We need to upgrade the frontend to reflect a modern, elegant, and highly trustworthy e-commerce platform. The goal is to make it user-friendly, bright, and convincing. You (Claude) have the creative freedom to apply standard UI/UX best practices to achieve a premium look.

Please execute the following tasks ONE BY ONE. Do not proceed to the next task until I approve the current one.

## Task 1: Global Theme & Color Palette Update

* **Default Theme:** Change the default theme from Dark to Light mode. The background should be a clean white (`#ffffff`) or a very soft off-white/gray (`#f9fafb`) to make product images pop.
* **Theme Switcher:** Implement a Dark/Light mode toggle switch in the header for users who prefer dark mode, but ensure Light mode is the absolute default.
* **Color Contrast:** In Light mode, ensure text is a sharp, highly visible black (e.g., `#111827` or `#000000`). Do not use faded or low-contrast text.
* **Buttons:** Make sure all Call-To-Action (CTA) buttons use a vibrant, premium accent color (like a rich Rose Red or Crimson) that stands out beautifully against the white background.

## Task 2: Professional Typography

* Replace the current "childish" or generic fonts with a highly professional, modern sans-serif font family such as `Inter`, `Montserrat`, `Poppins`, or `Plus Jakarta Sans`.
* Update the font weights: use bold/extrabold for headings and prices, and a clean regular weight for descriptions. Ensure the font sizes are readable across all mobile devices.

## Task 3: Header, Navigation & Search Functionality

* **Header Structure:** Upgrade the header to include the Brand Logo (left), Navigation Links (center), and Search/Theme Icons (right).
* **Interactive Search:** Implement a Search icon in the header. When clicked, it should open a top-down or center screen Modal/Overlay.
* **Search Logic:** As the user types, it should filter products. Clicking a search result should redirect them directly to that specific Product Details page.

## Task 4: Expanded Professional Footer & Informational Pages

* **Large Footer:** Remove the current minimal footer. Build a multi-column footer similar to standard e-commerce sites:
* *Column 1:* Brand Name, short tagline, and Social Media icons.
* *Column 2:* Quick Links (Home, Shop, Categories).
* *Column 3:* Legal & Info (About Us, Contact Us, Privacy Policy, Terms & Conditions).
* *Column 4:* Contact details (Dar es Salaam, WhatsApp Number, Email).


* **Static Pages:** Create the basic routing and elegant, readable placeholder pages for `About Us`, `Contact Us`, and `Privacy Policy`.

## Task 5: Home Page & Product Cards Refinement

* **Hero Section:** Improve the main headline area ("Urembo Halisi...") to look more like a premium banner with a clear "Shop Now" button.
* **Product Cards:** Enhance the product cards with subtle borders or very soft drop shadows (`shadow-sm`, hover `shadow-md`). Ensure the images fit perfectly, the pricing is large and clear, and the Like/Heart button is elegantly positioned, ensure allow to like and unlike on the same click.
